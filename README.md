# YGPanic by YG_WORKS

A standalone, duress alarm (Panic Button) script to work with FiveM.

- Integrated Discord embed for active unit status
- Avoids conflict with Sonoran scripts (Sonoran's panic made redundant)
- Unit status UI in game
- Per-department panic triggers (eg. fire won't recive ems or leo pings)
- Customizable unit logo's
- Integratable with [**YGCommands**](https://github.com/YG-WORKS/YGCommands)
- /toggleunits command (and keybind) to toggle unit status UI
- /duress command to trigger panic button (also keybind)
- /duresslogout to stop receiving alerts.
